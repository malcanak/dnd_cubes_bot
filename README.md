# 🎲 D&D Dice Bot

Телеграм-бот для бросания кубиков в D&D. Каждый игрок настраивает свои характеристики и комбо, всё управляется кнопками.

## Возможности

- 🎲 Бросать любые кубики (d4, d6, d8, d10, d12, d20, d100)
- ➕ Добавлять именованные характеристики с модификатором (например: «Атака молота» — d20+5)
- ⚔️ Создавать комбо из нескольких характеристик (например: «Удар» = попадание + урон)
- 🗑 Удалять характеристики и комбо
- 📋 Смотреть свои настройки
- Данные каждого игрока хранятся отдельно по Telegram ID

## Установка и запуск

### 1. Получи токен бота

1. Открой [@BotFather](https://t.me/BotFather) в Telegram
2. Напиши `/newbot`
3. Следуй инструкциям, скопируй токен

### 2. Локальный запуск (для теста)

```bash
pip install -r requirements.txt
BOT_TOKEN=ваш_токен python bot.py
```

### 3. Деплой на Railway

1. Залей папку на GitHub (репозиторий)
2. Зайди на [railway.app](https://railway.app) → New Project → Deploy from GitHub
3. В настройках проекта добавь переменную окружения:
   - `BOT_TOKEN` = твой токен от BotFather
4. Готово! Бот работает 24/7

## Структура данных

Данные хранятся в `data.json` в формате:

```json
{
  "123456789": {
    "stats": {
      "Атака молота": {"dice": "d20", "modifier": 5},
      "Урон молота": {"dice": "d8", "modifier": 3}
    },
    "combos": {
      "Удар молота": ["Атака молота", "Урон молота"]
    }
  }
}
```

## Использование

1. `/start` — открыть главное меню
2. Нажми «➕ Добавить характеристику» → введи название → выбери кубик → введи модификатор
3. Нажми «🔗 Добавить комбо» → введи название → отметь нужные характеристики
4. Бросай через «🎲 Бросить кубик» или «⚔️ Комбо-бросок»
